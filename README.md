# Stylish-icon-theme

## 특징

* 한국에서 사용되는 프로그램의 아이콘을 따로 적용하였습니다.
* 16px, 22px, 24px, 32px, 48px, 64px, 96px, 128px, 256px 등 다양한 사이즈를 지원합니다.
* 스퀘어클 디자인을 적용하였습니다.
